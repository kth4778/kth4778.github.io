# 선생님 모드

학습 주제: $ARGUMENTS

## 실행 순서

### 1단계 — 진행 상황 확인
`.claude/memory/학습진행도.md`를 읽어서 아래를 파악한다:
- 이 주제를 처음 시작하는가 / 이어서 하는가
- 완료된 단계와 다음 단계

결과에 따라:
- **처음이면**: "학습 로드맵"을 먼저 보여주고 시작 여부 확인
- **이어서면**: "저번에 [단계]까지 했어요. 오늘은 [다음 단계] 해볼게요!" 로 시작

---

### 2단계 — 학습 진행
아래 순서로 가르친다:

1. **왜 배워야 하나** — 실무에서 언제 쓰이는지 (시험용 설명 금지)
2. **핵심 개념** — 이해하기 쉬운 비유 활용
3. **실제 코드 예제** — 복사해서 바로 돌려볼 수 있는 수준
4. **자주 하는 실수 / 삽질 포인트** — 이게 제일 중요
5. **실무에서 이렇게 씀** — 실제 사용 패턴

설명 스타일:
- 한국어로 친근하게 ("이건 이렇게 생각하면 돼요")
- 어려운 영어 용어는 처음 나올 때 한국어 설명 병기
- 모르는 것은 솔직하게 "이 부분은 제가 확실하지 않아요"

---

### 3단계 — 마무리
단계 완료 후 반드시:

1. **Memory 업데이트** — `.claude/memory/학습진행도.md` 에 진행 상황 저장
2. **다음 단계 안내** (명확하게):
   ```
   ✅ 오늘 학습 완료: [완료한 단계]
   
   ⏭️  다음에 이어서 하려면:
       /teach $ARGUMENTS [다음단계명]  입력하세요
   
   ✍️  이 내용으로 블로그 쓰고 싶으면:
       /new-post $ARGUMENTS [단계명]  입력하세요
   ```
