# 현황 브리핑

아래 파일들을 읽고 현황을 정리한다:
- `.claude/memory/학습진행도.md`
- `.claude/memory/블로그현황.md`

아래 형식으로 출력:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 현재 진행 현황

📚 학습
  [기술명1] — [단계]
    ✅ 완료: [단계들]
    ⏭️  다음: [다음 단계]
    👉 이어서: /teach [기술명] [다음단계]

  [기술명2] — [단계]
    ...

✍️  블로그
  발행 완료: [N]편
  초안 대기: [제목 목록 또는 "없음"]

📌 추천 다음 작업
  1️⃣  [가장 먼저 해야 할 것]
  2️⃣  [그 다음]
  3️⃣  새로운 주제 시작 → 배우고 싶은 기술 말씀해주세요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
